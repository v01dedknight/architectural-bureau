# Архитектурное бюро - Сайт-визитка

Минималистичный современный сайт-визитка для архитектурного бюро. Проект ориентирован на визуальную подачу работ и удобство навигации.

## Структура проекта
Сайт состоит из двух основных страниц:
* **Главная (`index.html`)** - презентация бюро, ключевые услуги и контактная информация.
* **Портфолио (`portfolio.html`)** - галерея реализованных проектов с акцентом на качественные изображения.

## Технологии
* **HTML5 / CSS3** - адаптивная верстка и стилизация.
* **JavaScript** - интерактивные элементы и анимации.
* **Tilda Assets** - использованы оптимизированные компоненты для работы форм и сеток.

## Особенности
* **Адаптивность:** Сайт корректно отображается на десктопах, планшетах и смартфонах.
* **Скорость:** Изображения оптимизированы для быстрой загрузки.
* **Анимации:** Плавные переходы и появления элементов для создания премиального ощущения.

## Как запустить локально
1. Клонируйте репозиторий:
   ```bash
   git clone [https://github.com/v01dedknight/architectural-bureau.git](https://github.com/v01dedknight/architectural-bureau.git)
   ```

2. Откройте файл index.html в любом современном браузере (chrome, firefox ...)

Официальный сайт архитектурного бюро Rowelltz Arch. Минималистичный, современный и интерактивный веб-ресурс, демонстрирующий портфолио бюро и предоставляющий удобные способы связи с командой.

---

## Основные страницы

### Главная (index.html) – презентация бюро, ключевые услуги, миссия и контактная информация.
### Портфолио (portfolio.html) – галерея реализованных проектов с упором на визуальное качество и детали.

---

## Технологии

- HTML5 / CSS3 – семантическая и адаптивная верстка.
- JavaScript (ES6+) – интерактивные элементы, модальные формы, анимации.
- Nginx – reverse proxy, HTTPS, кэширование и безопасность.
- FastAPI + Uvicorn – backend-сервис для приёма заявок с сайта и маршрутизации данных.
- Python + HTTPX + dotenv – асинхронная передача данных во внешний сервис обработки заявок.
- Внешний сервис (Node.js / Telegram Bot Service) – отдельный backend, отвечающий за бизнес-логику и обработку заявок.

---

## Особенности проекта

- адаптивный дизайн. Полноценная поддержка мобильных устройств, планшетов и настольных ОС.
- оптимизация скорости. Изображения и скрипты минимизированы для быстрой загрузки.
- интерактивные элементы. Плавные анимации, модальные формы и динамическое отображение контента.
- безопасность. CORS настроен корректно, HTTPS через nginx, токены и секреты хранятся в .env.
- модульная backend-архитектура. Backend разделён на сервис приёма заявок и отдельный сервис бизнес-логики.
- автономный backend. FastAPI-сервис работает как служба systemd и функционирует независимо от ssh-сессий.
- сервисная интеграция. Данные с сайта передаются во внешний backend-сервис для дальнейшей обработки.

## Установка и локальный запуск

1. Клонируйте репозиторий:
``` bash
git clone https://github.com/v01dedknight/architectural-bureau.git
cd rowelltz-arch
```

2. Создайте виртуальное окружение и установите зависимости:
``` bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

3. Создайте .env с токеном бота и chat_id:
``` css
BOT_TOKEN=<ваш_telegram_bot_token>
CHAT_ID=<ваш_chat_id>
```

4. Запустите сервер локально:

``` bash
python server.py
```

Откройте *http://127.0.0.1:8000* в браузере, чтобы протестировать формы.

---

## Продакшен

- backend работает через systemd unit (rowelltz-api.service).
- nginx проксирует /api/send на FastAPI.
- HTTPS обеспечен сертификатом Let’s Encrypt.
- FastAPI принимает заявки с сайта и передаёт их во внешний сервис обработки заявок.
- внешний сервис отвечает за бизнес-логику и интеграцию с Telegram-ботом.

## Аналитика и сбор данных

На сайте подключена Google Analytics (GA4) для сбора аналитических данных:

- посещаемость сайта
- источники трафика
- поведение пользователей
- время на страницах
- конверсии форм
- взаимодействие с интерфейсом

Аналитика используется для:
- улучшения пользовательского опыта
- оценки эффективности страниц
- оптимизации структуры сайта
- принятия продуктовых решений

Сбор данных осуществляется через официальный тег Google Analytics, встроенный в `<head>` страниц сайта.
